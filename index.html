Sunday, February 5th 2017 - Friday, February 10th 2017

Now I’ve started for real!

For the last week I’ve been working in the office of North Kingdom. My main focus for this week has been to get everything set up (tool for writing report, git repo and logbook solution). I’ve also been reading a lot, and I mean A LOT of scientific papers on VR in general, interaction and interfaces in VR and most of all about tools and techniques that are used for interacting in VR. I try to only read papers that I can use to back my method and decisions on, even though my subject and focus is quite abstract and unclear at the moment.

My original direction for the thesis has been about evaluating design principles used for screens when applied to VR. My assumption was that those principles were used in the majority of applications today, but based on the articles I’ve been reading the industry has moved past this already. I’ve shifted more and more towards reading about selection and interaction tools. This is an interesting subject that has been researched a lot in the past, which have so far presented me with the following:

  • Interaction tools and techniques are strongly dependant on the context of the application.
  • Using a ray (laserpointer) to interact is prefered over a “simulated hand” when interaction with menus that are sparse with few sizable objects. In a more dynamic environment that requires more precision different pointer-techniques are better.
  • Using a simulated hand is really intuitive and provides an option for delicate and precise interactions.

Next week I’m going to (hopefully) be briefed on the HBO project and set a context for my research, which would help tremendously. Until then I’ll keep catching up on the latest reseach, I’ve been working my way up from 1992 to 2012 so far.
